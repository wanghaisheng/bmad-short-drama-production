# Novel → Short Drama Adaptation Checklist

## Source Readiness
- [ ] Synopsis available and coherent
- [ ] Key arcs identified (setup / rising / midpoint / climax / resolution)
- [ ] Character list with roles and conflicts

## Arc & Episode Mapping
- [ ] Chapters mapped to 60–100 episodes
- [ ] Each episode has a hook and cliffhanger slot
- [ ] Reversals placed at block milestones (e.g., Ep5/Ep10)

## Scene Compression
- [ ] Descriptive prose converted to visual beats
- [ ] Dialogue shortened to punchy lines
- [ ] Visible conflict in every scene

## Hook Candidates
- [ ] First frame is high-contrast and action-centric
- [ ] Burning question posed within 3 seconds
- [ ] Audio cue planned (sfx or music stop)

## Conflict Density
- [ ] 3–5 micro-conflicts per episode
- [ ] Stakes escalated (personal/public/irreversible)
- [ ] Emotional outbursts visible, not implied

## Cliffhanger Anchors
- [ ] 85% episode completion point used for suspense
- [ ] Choice of suspense/reversal/cool-point
- [ ] Next-episode curiosity gap clear

## Visual Consistency
- [ ] Character references defined
- [ ] Shot language includes motion and lighting
- [ ] Vertical composition centered and clean
