## üöÄ Installation

### Via BMad Installer (After PR Acceptance)
```bash
npx bmad-method install
# Select "Short Drama Pack" from the expansion packs list
```

### Manual Installation
1. Clone or download this expansion pack
2. Copy to your BMad Method installation:
   ```bash
   cp -r vidu-story-writing/* ~/bmad-method/expansion-packs/vidu-story-writing/
   ```
3. Run the BMad installer to register the pack
